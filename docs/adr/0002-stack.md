# ADR-0002. Стек технологий

- **Статус:** принято (Фаза 0/1, 2026-07-06; источник — мастер-промпт v2, «Архитектурные решения» №2)
- **Контекст.** Шаблон тиражируется по клиентским проектам; каждая экзотическая зависимость умножает стоимость поддержки на число проектов.
- **Решение.** Python 3.12+, FastAPI, SQLAlchemy 2.x (async) + Alembic, PostgreSQL 16, Redis (кеш + очереди arq), Docker Compose. Пакетный менеджер — uv (лок-файл — часть supply-chain-защиты, V8). Наблюдаемость: structlog (JSON), Sentry (DSN из env), prometheus-client. Только зрелые популярные библиотеки; каждая новая зависимость требует обоснования в PR.
- **Последствия.** Воспроизводимые сборки (uv.lock, `uv lock --check` в CI); единый стек во всех клиентских проектах; апгрейды стека приезжают релизами шаблона (ADR-0009).
